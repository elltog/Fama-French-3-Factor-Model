# Fama-French-3-Factor-Model

This project analyzes the performance of a U.S. mutual fund using the Fama-French Three-Factor Model and compares it conceptually with the CAPM framework.
Monthly fund returns are constructed from daily adjusted closing prices obtained via Yahoo Finance. These returns are then converted into excess returns and regressed on the Fama-French risk factors to examine the fund’s exposure to systematic market, size, and value risks.

Fund price data: Yahoo Finance

Fama-French factors: Kenneth R. French Data Library (https://mba.tuck.dartmouth.edu/pages/faculty/ken.french/data_library.html)
